# TreeStruct_Cangjie

This is a tree-based data structure project developed in the Cangjie language.  
It includes multiple tree implementations, such as general trees, binary trees, binary search trees, and more. Additional features may be added in future updates.

## Project Introduction

This project encapsulates tree-related data structures into classes for easier learning, use, and extension.  
It provides common tree operations, including:

- Element insertion
- Element deletion
- Multiple tree traversal methods
- Tree-based sorting function encapsulation

## Supported Structures

Currently, the project may include the following tree structures:

- Tree
- Binary Tree
- Binary Search Tree (BST)

More tree structure implementations may be added in future versions.

## Features

- Developed based on the Cangjie language
- Encapsulated in class-based form for easier use and maintenance
- Supports element insertion and deletion
- Supports multiple traversal methods, such as:
  - In-order traversal
  - Pre-order traversal
  - Post-order traversal
  - Level-order traversal
- Includes some sorting function implementations based on tree structures
- Suitable for learning, practice, and reference
